# Remember Me 프로젝트

## 📖 프로젝트 설명

다양한 수단들로 암기를 시키는 앱 서비스.
1단계 목표는 장소를 이용해 암기를 트리거 시킨다.

## 💿 개발 환경 및 스택

React.js / TS / Parcel

## 🔗 배포링크

[🔗 배포 링크 이동](https://size-to-size.vercel.app//)

## 💻 실행방법

```bash
# 프로젝트 클론
git clone "https://github.com/2taesung/SizeToSize.git"

# 프로젝트 폴더 진입
cd sizetosize

# 모듈 설치 및 실행
yarn install && yarn dev
```
