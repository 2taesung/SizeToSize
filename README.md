# Remember Me 프로젝트

## 📖 프로젝트 설명

내가 가지고 있는 신발의 사이즈로 내가 사고 싶은 신발의 사이즈를 통계적인 증거를 제시해 추천

## 💿 개발 환경 및 스택

Next.js / TS / yarn

## 🔗 배포링크

[🔗 배포 링크 이동](https://size-to-size.vercel.app//)

## 💻 실행방법

```bash
# 프로젝트 클론
git clone "https://github.com/2taesung/SizeToSize.git"

# 프로젝트 폴더 진입
cd sizetosize

# 모듈 설치 및 실행
yarn install && yarn dev
```
